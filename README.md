# Drug Discovery

# Project Description 
## Introduction
The goal of this project was to train a model that could predict whether a compound would be active against a target protein. The model was trained using the inhibition activity of more than 5,000 compounds towards a target protein (gamma-secretase). In an effort to discover more drugs for Alzheimer's disease, in silico characterization of compounds for possible activity is crucial. 

## Significance
This approach saves time and money. Only the most promising compounds will advance to wet lab experiments.
